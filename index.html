STEP 01 / 05

Grab an Android Device

Any Android phone or tablet running Android 11+ will work. The Jugaad J1 runs on pure Android — no special hardware required.

I have an Android device (phone or tablet) required
Device is running Android 11 or newer recommended
Device is charged and connected to WiFi required
▶ Having issues?
My device is too old / below Android 11
The J1 can still work on Android 10, but some features may be limited. Try step 4 first — if the Assistant option isn't there, you may need a newer device.
My device won't turn on
Hold the power button for 15+ seconds to force a restart. If that fails, charge for 30 minutes and try again.
No Google Play Store
Some devices (e.g. Huawei) lack Play Services. You'll need to sideload APKs from trusted sources like APKMirror.
STEP 02 / 05

Install Apps

Open the Play Store and install each app. Check them off as you go:

Indistract Launcher launcher
Manus AI Agent agent
Perplexity AI Assistant assistant
Shake It Gesture Launcher gesture
▶ Having issues?
App not found in Play Store
The app may not be available in your region. Try searching the exact name or download the APK from the developer's website.
"Device incompatible" error
Some apps need specific sensors (e.g. Shake It needs an accelerometer). Check app requirements in the Play Store listing.
Download stuck / won't install
Clear the Play Store cache (Settings → Apps → Play Store → Storage → Clear Cache) and restart the download.
STEP 03 / 05

Configure Accounts & Launcher

Create an account on Manus AI account
Create an account on Perplexity account
Set Indistract as the default launcher
Settings → Apps → Default Apps → Home App → Indistract launcher
Inside Manus: set up Skills, MCP Servers, and Connectors advanced
▶ Having issues?
Can't create an account
Make sure you're using a valid email. Some services may require a Google or Apple sign-in — try both options.
Indistract doesn't show up as a launcher option
Open Indistract once and complete its initial setup. Then go to Settings → Default Apps and it should appear.
Manus setup (Skills/MCP) is confusing
Start with the default skills — you can always add more later. MCP servers and connectors are optional for advanced use.
STEP 04 / 05

Enable Perplexity Assistant

Open Perplexity → tap the gear icon step
Scroll down to Assistant and tap it step
Enable the assistant and follow the on-screen steps step
Test: hold the power button → Perplexity should open test
▶ Having issues?
Can't find "Assistant" in settings
Make sure Perplexity is updated to the latest version. If it's still missing, your device may not support assistant features — try Android 12+.
Power button doesn't trigger the assistant
Go to Settings → Apps → Default Apps → Digital Assistant App and select Perplexity. Power button should then work.
Assistant says "not available in your region"
Try using a VPN set to a supported country (US or UK). You can disable it after setup is complete.
STEP 05 / 05

Configure Shake It

Shake It lets you launch Manus with a simple shake gesture.

Open Shake It app step
Grant all required permissions
(Notification access, Draw overlays, Background launch) permissions
Set the action to open → Manus action
Test: shake your device → Manus should launch test
▶ Having issues?
Permissions don't stick / keep revoking
Some Android versions auto-revoke permissions. Go to Settings → Apps → Shake It → Permissions and re-enable them. Disable battery optimization for Shake It too.
Shaking doesn't do anything
Make sure Shake It's service is running (check the app's status). Try a more vigorous shake or adjust sensitivity in Shake It settings.
Wrong app opens when I shake
Double-check the target app selection in Shake It. Make sure you picked Manus and not another app.
